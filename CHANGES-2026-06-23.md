## Changes Made

### 1. `minilisp.py` - Code Fixes

#### `Env.find()` method (lines 95-102)
**Before:**
```python
def find(self, var):
    return self if var in self else self.outer.find(var)
```
**After:**
```python
def find(self, var):
    if var in self:
        return self
    elif self.outer is not None:
        return self.outer.find(var)
    else:
        raise NameError(f"Name '{var}' is not defined")
```
**Reason:** Fixed crash when variable not found in any environment. Now raises clear `NameError` instead of `'NoneType' object has no attribute 'find'`.

#### `parse()` function (lines 41-56)
**Added:**
```python
elif token == "'":
    # Handle quote shorthand: 'x or '(...
    if not tokens:
        raise SyntaxError("Unexpected end of input after quote")
    next_token = tokens.pop(0)
    if next_token == '(':
        # ' followed by ( - quote a list
        tokens.insert(0, '(')
        expr = parse(tokens)
        return ['quote', expr]
    else:
        # ' followed by atom - quote the atom
        return ['quote', atom(next_token)]
```
**Reason:** Handles quote shorthand for cases like `'(` (where quote and paren are separate tokens).

#### `atom()` function (lines 63-65)
**Added:**
```python
# Handle quote shorthand for tokens like 'yes (single token with quote prefix)
if token.startswith("'"):
    return ['quote', token[1:]]
```
**Reason:** Handles quote shorthand for cases like `'yes` (where quote is attached to the token).

    #### `repl()` function (lines 390-413)
    **Before:**
    ```python
    if line.strip().lower() in ('exit', 'quit'):
        print("Goodbye!")
        break
    
    if line.strip().lower() in ('help', '?'):
        print_help()
        continue
    
    # Tokenize and parse
    tokens = tokenize(line)
    ast = parse(tokens)
    ```
    
    **After:**
    ```python
    # Tokenize and parse first to handle both (exit) and exit
    tokens = tokenize(line)
    ast = parse(tokens)
    
    # Check for special commands - handle both bare words and (command) forms
    if isinstance(ast, str):
        if ast.lower() in ('exit', 'quit'):
            print("Goodbye!")
            break
        elif ast.lower() in ('help', '?'):
            print_help()
            continue
    elif isinstance(ast, list) and len(ast) == 1:
        if ast[0].lower() in ('exit', 'quit'):
            print("Goodbye!")
            break
        elif ast[0].lower() in ('help', '?'):
            print_help()
            continue
    ```
    **Reason:** REPL now handles both `exit` and `(exit)` forms, `help` and `(help)` forms. Previously only bare words worked.

## New Feature: Pretty Printing (pprint)

Added a `pprint` function that pretty-prints Lisp expressions in classical Lisp format with proper indentation.

### Implementation

1. **`minilisp.py`** - Added `pprint()` function and registered it as a built-in
   - Recursively converts Python data structures to Lisp-style strings
   - Handles: atoms (numbers, symbols), booleans (#t/#f), empty lists (()), simple lists, nested lists
   - Uses 2-space indentation for nested structures
   - Simple lists (all atoms, ≤5 elements) print inline: `(1 2 3)`
   - Nested lists print with each element on its own line, properly indented
   - Closing parentheses align with opening parentheses

2. **`tests/test_minilisp.py`** - Added `TestPPrint` class with 12 tests:
   - Atom printing (integers, floats, symbols, booleans)
   - Empty list handling
   - Simple list formatting
   - Nested list indentation
   - Deeply nested structures
   - Integration through evaluator

3. **`EXAMPLES.md`** - Added new "Pretty Printing" section:
   - Shows usage examples for pprint
   - Documents formatting conventions
   - Updated table of contents

4. **`demo.lisp`** - Added pprint examples at the end

5. **`.claude/canvas/gist.yml`** - Updated features list:

### Example Usage

```lisp
minilisp> (pprint '(1 2 3))

minilisp> (pprint '(+ 1 (* 2 3)))

```
